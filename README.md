## 模块总览

### 语言基础

从零搭建 C 语言基本功，涵盖完整语法体系。

- **文件位置**：`Three_up/`（按日期子目录）
- **时间跨度**：11月 ~ 次年1月
- **核心主题**：
  - 数据类型与运算符
  - 循环与分支控制流
  - 位运算
  - 指针（基础到函数指针）
  - 字符串处理（含大数加减法实战）
  - 动态内存管理
  - 结构体与类型定义
  - 函数与宏

> 如需回顾某个主题，按子目录名称中的日期定位即可。

### 数据结构与算法

用 C 实现经典数据结构与基础算法，强调手写而非调库。

- **文件位置**：`Three_down/` 中 03~04 月份的子目录
- **核心主题**：
  - 线性结构：顺序表、单向/双向/循环链表、栈、队列、循环队列
  - 树形结构：二叉搜索树（插入、遍历、翻转）
  - 图形结构：DFS、BFS
  - 散列结构：线性探测哈希表、哈希链表
  - 堆：堆的构建与操作
  - 字符串算法：BF 暴力匹配、KMP 算法
  - 经典问题：斐波那契数列

### 系统编程

深入操作系统层面的编程能力，涵盖并发控制与底层资源管理。

- **文件位置**：`Three_down/` 中 05~06 月份的子目录
- **核心主题**：
  - 文件系统：文件读写、二进制文件处理、目录遍历
  - 并发编程：线程创建与回收、生产者-消费者模型
  - 同步机制：互斥锁、读写锁、自旋锁、信号量、条件变量
- **涉及头文件/库**：`pthread.h`、`semaphore.h`、`dirent.h`

### 网络与通信

基于 Socket API 的网络编程实践。

- **文件位置**：`Three_down/` 中 07 月份的子目录
- **核心主题**：
  - TCP 客户端/服务端模型
  - 网络字节序与地址转换
- **涉及头文件/库**：`sys/socket.h`、`netinet/in.h`、`arpa/inet.h`

### 密码学与编码

使用实用库进行加密与编码操作。

- **文件位置**：`Three_down/0531摘要算法/`、`Three_down/0605base64/`

## 如何添加新内容

本仓库采用模块化组织方式，添加新代码只需遵循以下规则：

### 1. 新建子目录

在对应学期目录下，按 `MMDD主题` 格式创建子目录：

```
Three_down/0715新主题/
    ├── demo.c          # 示例代码
    └── 笔记.md          # 学习笔记（可选）
```

### 2. 更新 README

根据新内容的类别，在「模块总览」对应章节追加简要说明；若引入全新领域，新建一个模块小节并同步更新「学习路线图」中的状态。

### 3. 命名规范

- 子目录：`MMDD主题`（如 `0715进程管理`）
- 源文件：小写英文 + 数字编号（如 `hw1.c`、`server.c`、`demo.c`）
- 笔记：中文描述性命名（如 `BIO学习记录.md`）

### 4. 代码风格建议

- 关键逻辑添加中文注释
- 涉及外部库编译时，在文件头部注释编译命令（如 `// gcc x.c -o x -lpthread`）
- 保持函数简短，单一职责

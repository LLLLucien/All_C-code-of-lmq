# All_C-code-of-lmq

C 语言学习过程中的代码记录仓库，按学期和日期组织。

## 目录结构

### Three_up / Three_down
大三上学期和大三下学期的学习代码，包含：
- **基础知识**：指针、结构体、动态内存、字符串、函数等
- **数据结构**：链表、栈、队列、哈希表、二叉搜索树、DFS/BFS、字符串匹配（KMP、BF）、斐波那契等
- **文件IO**：文件读写操作
- **目录操作**：目录遍历和管理
- **二进制文件**：二进制数据处理
- **Base64编码**：编码学习

### WiVaPractice
课本练习代码

### 简历
简历相关教程

## 已独立项目

以下项目已分离为独立仓库：
- [ShortURL_C](https://github.com/LLLLucien/ShortURL_C) - 短网址服务
- [Elevator-C](https://github.com/LLLLucien/Elevator-C) - 电梯调度算法

## 说明

代码按日期命名（如 `0321链表` 表示 3 月 21 日的链表学习），方便回顾学习进度。
大部分为课程学习中的练习代码，包含数据结构和算法实现。

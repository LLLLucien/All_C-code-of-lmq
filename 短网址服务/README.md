# 短网址服务

基于哈希算法实现的短链接生成工具，支持有效期管理和访问统计。

## 功能特性

| 功能 | 说明 |
|------|------|
| 生成短链接 | 将长 URL 转换为 6 位短码 |
| 有效期管理 | 支持 1天 / N天 / 永久 三种模式 |
| 链接查询 | 通过短链接反向查询原始 URL |
| 统计面板 | 查看解析次数、访问次数、创建时间 |

## 算法原理

1. **哈希计算**：使用 Times33 算法将长 URL 转为唯一整数
2. **Base62 编码**：将整数转换为 62 进制（a-z, A-Z, 0-9）
3. **数组存储**：使用动态数组存储短链接与原始链接的映射关系

## 使用方法

### 编译

```bash
gcc -o Main main.c ShortURL.c -std=c99
```

### 运行

```bash
./Main
```

## 核心函数

| 函数 | 功能 |
|------|------|
| `Turn_to_hash()` | Times33 哈希算法 |
| `hash_to_base62()` | 哈希值转 Base62 编码 |
| `InputURL()` | 长链接输入 |
| `parse_short_url()` | 短链接解析查询 |

## 技术栈

- **语言**：C
- **存储**：内存数组（无数据库）
- **编码**：Times33 哈希 + Base62

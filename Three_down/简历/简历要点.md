# 简历撰写要点

---

## 一、基本信息

| 项目 | 要点 |
|------|------|
| **标题** | 姓名 + 应聘职位（不要写"个人简历"） |
| **联系方式** | 年龄、性别、手机号、邮箱、籍贯 |
| **邮箱选择** | 推荐 `libai.cs@outlook.com`，避免 QQ 邮箱 |
| **附加链接** | 个人博客、GitHub、作品集（能直接访问展示能力） |

**注意**：邮箱设置为可点击链接，方便HR直接发送邮件。

---

## 二、教育背景

| 项目 | 要点 |
|------|------|
| **核心信息** | 学校、专业、学历、在校时间 |
| **亮点加分** | GPA（如3.5+/4.0）、排名（如专业前10%） |
| **证书荣誉** | 四六级、专业证书、奖学金、竞赛奖项 |
| **备选内容** | 核心专业课名称（当其他内容较少时） |

---

## 三、专业技能

### 技能分类建议：

| 类别 | 示例 |
|------|------|
| **编程语言** | C/C++、Python、Java、Go |
| **开发框架** | React、Vue、Spring Boot |
| **数据库** | MySQL、Redis、MongoDB |
| **工具技能** | Git、Docker、Linux、VS Code |
| **其他能力** | 英语能力、项目管理、文档撰写 |

**技巧**：每项技能可标注熟练度（如"熟练"、"掌握"、"了解"），并简要举例说明应用场景。

---

## 四、项目经历

### 项目信息结构：
1. **项目名称** + **项目周期** + **技术栈**
2. **项目描述**：项目背景、目标、规模
3. **职责与贡献**：
   - 负责模块/功能
   - 使用的技术方案
   - 解决的核心问题
   - 取得的成果（量化优先）

**STAR法则**：
- **S**ituation（背景）
- **T**ask（任务）
- **A**ction（行动）
- **R**esult（结果）

**示例**：
> 使用 C 语言实现 Base64 编解码工具，解决二进制文件加密传输问题，提升数据传输安全性。

## 简历撰写小贴士

1. **简洁清晰**：控制在1-2页，重点突出
2. **量化成果**：用数字说明贡献（如"优化算法性能提升30%"）
3. **关键词匹配**：根据岗位JD调整内容
4. **格式规范**：统一字体、间距、对齐方式
5. **语法检查**：避免错别字和语法错误
